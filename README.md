# Othera

Othera® — Biphasic Esomeprazole product website for ACI Limited.

Built with **Vite + React + React Router + Framer Motion**.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Production build

```bash
npm run build
npm run preview
```

The static output is written to `dist/`.

## Pages

| Route | Page |
|-------|------|
| `/` | Home |
| `/about` | About Othera |
| `/products` | Product range |
| `/why` | Why Biphasic |
| `/gerd` | GERD Education |
| `/faq` | FAQs |
| `/hcp` | Healthcare professionals |
| `/contact` | Contact |
